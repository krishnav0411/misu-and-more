# Misu & More

A premium handcrafted dessert website built with Next.js, TypeScript and Tailwind CSS.

## Live Demo

https://misu-and-more.vercel.app

## Features

- Responsive design
- Elegant landing page
- Shop page
- Dynamic product pages
- Smooth navigation
- Mobile friendly

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- Framer Motion

## Run Locally

```bash
npm install
npm run dev
```

Then open:

```
http://localhost:3000
```
